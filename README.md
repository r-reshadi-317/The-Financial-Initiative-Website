# The Financial Initiative Website

Website for The Financial Initiative Nonprofit

## Pages

- **Home** — Hero, impact stats, trusted partners
- **Programs** — Five workshop programs + custom session CTA
- **About** — Mission, team cards, partner organizations
- **Get Involved** — Ways to participate + contact form (opens mail client)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Design notes

Colors and layout match the original Base44 site: navy gradients (`#0a1628`, `#0d2347`, `#1a3a6b`), accent cyan (`#17a2d8`), and Inter typography. Partner logos load from the original CDN.
